# Sales-Analysis-Dashboard

## Project Overview

This project focuses on analyzing sales performance and business insights using Microsoft Excel and Power BI. The dashboards help identify sales trends, profitable categories, customer segments, regional performance, and shipping efficiency.

## Excel Dashboard

### Features

* KPI Cards (Sales, Profit, Quantity, Orders)
* Interactive Region Filter
* Sales Analysis by Product
* Sales Analysis by Customer
* Regional Performance Analysis
* Business Insights Section

### Purpose

The Excel dashboard provides a quick overview of business performance and helps identify best-selling products, top customers, and regional sales trends.

---

## Power BI Dashboard

### Features

* KPI Cards (Total Sales, Profit, Quantity, Orders)
* Interactive Slicers (Region, Segment, Category, Ship Mode)
* Sales by Category
* Sales by Region
* Profit by Segment
* Profit by Ship Mode
* Profit by Sub-Category
* Year-wise Trend Analysis

### Purpose

The Power BI dashboard provides interactive business intelligence and helps analyze profitability, customer segments, regional performance, and business growth over time.

---

## Tools Used

* Microsoft Excel
* Power BI Desktop
* Data Visualization
* Data Analysis

## Key Insights

* Identified high-performing categories and sub-categories.
* Compared regional sales performance.
* Evaluated shipping mode profitability.
* Analyzed customer segments.
* Tracked business growth using trend analysis.
 
## Google Looker Studio Dashboard

Created an interactive sales analytics dashboard using Google Looker Studio.

### Features

* KPI Cards (Sales, Profit, Quantity, Profit Margin)
* Interactive Filters (City, Region, Year)
* Region-wise Sales Analysis
* Quantity by Category
* Profit by Ship Mode
* Profit by Year
* State-wise Sales Analysis

### Tools Used

* Google Looker Studio
* CSV Dataset
* Data Visualization
* Dashboard Design

### Outcome

Built an interactive business dashboard that enables users to explore sales performance through dynamic filtering and visual analytics.
